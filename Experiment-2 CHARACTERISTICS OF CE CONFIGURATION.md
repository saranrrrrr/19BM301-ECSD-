# Experiment-2 CHARACTERISTICS OF CE CONFIGURATION
# Aim:
To Plot the input and output characteristics of the transistor in CE mode, to determine the h-parameters from the graph.
# Components Required:
<img width="681" height="339" alt="image" src="https://github.com/user-attachments/assets/4260154b-688f-4e0f-a430-8f90b498072f" />
# Procedure:
INPUT CHARACTERISTICS
1. Connections are made as per the circuit diagram.
   
2. VCE is kept const (say 5v), VBE is varied insteps of 0.1v and the corresponding IB values are
tabulated.

3. The above procedure is repeated for different constant values.

4. Graph is plotted between VBE and IB for a constant VCE
   
5. The h – parameter values are found from the graph.
   
OUTPUT CHARACTERISTICS

1. Connection are made as per the circuit diagram
2. IB is kept const, VCE is varied in step of 1V, the corresponding IC values are tabulated. The
above procedure is repeated for different IB constant values.
3. Graph is plotted between VCE and Ic for a constant IB.
4. The h – parameter values are found from the graph.
